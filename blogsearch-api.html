1. 블로그 검색

1.1. [성공] 키워드 이용 블로그 검색

1.1.1. 요청

쿼리 파라미터
이름 설명 필수여부

query

검색을 원하는 질의어

O

sort

검색 기준, accuracy(정확도) 또는 recency(최신순), 기본값 accuracy

X

page

결과 페이지 번호, 1 ~ 50의 수, 기본값 1

X

size

한 페이지에 보여줄 결과의 수, 1 ~ 50사이의 값, 기본값 10

X
요청예시
GET /api/blog/search?query=bankok&sort=accuracy&page=1&size=10 HTTP/1.1
Host: localhost:8080

1.2. [실패] 키워드 미입력/공백

1.2.1. 요청

키워드 미입력 또는 공백

요청예시
GET /api/blog/search?query&sort=accuracy&page=1&size=10 HTTP/1.1
Host: localhost:8080

1.2.2. 응답

응답 본문
이름 타입 설명

code

String

code 번호 : Presentation - Validation - 001

message

String

오류 메시지

messageDetail

String

추가 오류 메시지
응답예시
HTTP/1.1 400 Bad Request
Content-Type: application/json
Content-Length: 104

{
  "code" : "P-V-001",
  "message" : "검색 쿼리는 필수값입니다.",
  "messageDetail" : null
}
